# Weather-Based-Prediction-of-Wind-Turbine-Energy-Output
This project predicts wind turbine electricity output using weather data like wind speed, temperature, and pressure. A machine learning model learns from past data to estimate future power generation, helping improve energy planning, reduce wastage, and make renewable energy more reliable.

## Project Setup

# 1. Clone the Repository- 
git clone https://github.com/sai446755-ai/Weather-Based-Prediction-of-Wind-Turbine-Energy-Output/tree/main/Wind%20Energy%20Prediction

# 2. Create & Activate Virtual Environment
python -m venv .venv
# Linux / Mac
source .venv/bin/activate
# Windows
.venv\Scripts\activate

# 3.Install Dependencies
<pip install -r requirements.txt>
If requirements file is missing:
<pip install tensorflow keras numpy pandas matplotlib opencv-python flask>

# 4.Download Dataset
Link:  <https://www.kaggle.com/datasets/paultimothymooney/blood-cells/data>
